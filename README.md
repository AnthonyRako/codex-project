# codex-project

This repository contains a small React/TypeScript project for calculating real estate investment metrics.

## Structure

- `package.json` and `tsconfig.json` configure the project.
- `public/index.html` is the HTML template.
- `src/index.tsx` bootstraps the React app.
- `src/InvestmentCalculator.tsx` implements the calculator logic.
- `src/App.tsx` renders the calculator.

## Usage

1. Run `npm install` to install the dependencies.
2. Start the development server with `npm start`.

The application will open in your browser and show a simple form where you can enter various property parameters and immediately see the calculated results.
